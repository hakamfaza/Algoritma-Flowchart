Deteksi Palindrom
1. Mulai.
2. Masukan text berupa string.
3. Jika text bukan string maka kembali ke tahap 2.
4. Membuat variable x = text.length - 1.
5. Jika variable x >= 0 maka buat variable hasil yang berisi nilai hasil + text[x].
6. Setiap terjadi looping , variable x akan - 1;
7. Jika variable x <= 0 maka akan melanjutkan kondisi selanjutnya.
8. Jika hasil = text, maka output adalah palindrom.
9. Jika hasil != text, maka output adalah bukan palindrom.
10. Selesai.
